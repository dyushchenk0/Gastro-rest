# Gastro Chef

**Gastro Chef** is an educational project — an online food delivery store, created as a course project. The main focus of the project is on visual design and working with a database.  

**Project Description**
The project allows users to:  
- Browse a list of dishes.  
- Work with a client database.  
- Send feedback through a contact form.  
- Switch the interface language using a database.  

The project was created to strengthen skills in React, Node.js, and database management.  

> ⚠️ Note: This is an educational project. Responsive design is minimal and currently only supports desktop.  

**Technologies**
- **Frontend:** HTML, CSS/SCSS, JavaScript, React, React Router, Vite  
- **Backend:** Node.js (basic), SQL, MongoDB  
- **Tools:** Git, GitHub, Figma, VS Code, WebStorm  

**Features**
- Interaction with databases through custom APIs.  
- Well-organized project structure and file hierarchy.  
- Extensive use of React JSX and SCSS.  
- The project is planned for future improvements if time allows.  

**Installation and Running**
1. Clone the repository:
```bash
git clone https://github.com/dyushchenk0/gastro-chef.git
```

2. Install frontend dependencies:
```bash
npm install
```

3. Run the frontend:
```bash
npm run dev
```
Open the project in your browser at the provided port (usually http://localhost:5173
)

4. Run the backend server:
```bash
сd api
npm start
```
